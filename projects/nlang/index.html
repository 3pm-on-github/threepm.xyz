nlang

nlang is a language consisted of not letters, but numbers. This page is all about this language, and also the official nlang translator.

How to learn nlang (w/ examples)

nlang is based on English. So you don't need to learn any words, you only need to remember the numbers that correspond to the letters. nlang does not have any ponctuation nor any lowercase letters or numbers. It only has uppercase letters. nlang is written with numbers because humans can remember numbers better than letters.

Here is how to learn nlang; it's pretty straightforward. First of all, an nlang letter is 2 numbers long. 00 is space. Numbers from 01-26 are letters from A-Z.

Example: HELLO WORLD -> 0805121215002315181204
